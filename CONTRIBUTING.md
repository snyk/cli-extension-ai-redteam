# Contributing

This repository is closed to public contributions.

All contributions are made by internal Snyk teams. If you have found a security issue, please see [SECURITY.md](SECURITY.md).

Any accepted contributions require signing the [Snyk CLA](https://gist.github.com/snyksec/201fc2fd188b4a68973998ec30b57686#file-snyk-oss-contributor-agreement-md).
